CSS Topic- Fonts!

This is a simple walk through about how to choose a font, bold or italicize words, and change the size of fonts in different ways.

Choose Your Font

Browsers are only able to display fonts that they have access to. CSS provides a pathway around this problem with the "font-family" tag. To use this tag, you specify which font you want the browser to display, and provide back up fonts that the browser can use as default options if it doesn't have access to the other preferred fonts.

Bolding

Fonts are bolded using the "font-weight" property. You can use this tool not only to bold fonts but to also control how heavily they are bolded.

Changing Font Size

There are many different ways to define a font's size. Different properties use various units to define sizes.These properties include:

  • ABSOLUTE: based on the font's default size (font-size:large, small, x-large, medium, etc.)
  • RELATIVE: increases or decreases font size relative to the parent element's font size (font-size: smaller, larger, etc.)
  • PERCENTAGE: changes the font size relative to its parent element's size (font-size: 80%, 120%, etc.)
  • LENGTH: computes font size as multipliers of the set default font sizes
    • font-size: 1.5em (relative to parent element)
    • font-size: 1.8rem (relative to root element)
    • font-size: 1.2ex (relative to the height of first available font on the page)

Pixels (px) are not often used because they do not provide much flexibility in design across platforms. For example, when adapting from a computer screen to a phone screen, ensuring the header remains 1.5 times larger than the text underneath it makes it much more readable on a phone than saying that the header always has to be "24 pt font", which will look a lot bigger on a phone screen than on a full size monitor.
